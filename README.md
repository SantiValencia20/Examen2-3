# Examen 2 - Proyecto Web con BEM y Bootstrap

Este es el proyecto final del examen web, maquetado de acuerdo a una imagen de referencia respetando estándares de diseño responsivo.

## Tecnologías y Metodologías usadas
* **HTML5**: Estructura semántica base.
* **CSS3**: Hoja de estilos principal.
* **Bootstrap 5**: Utilizado principalmente para las grillas (`row`, `col`) y componentes básicos como el `navbar` que garantizan que el sitio sea Responsive en celulares, tablets y PC.
* **Metodología BEM**: Se aplicaron clases estructuradas por Bloque, Elemento y Modificador (ej. `hero-section__title`, `topbar__link`) para facilitar el mantenimiento.

## Despliegue
El proyecto está alojado de forma gratuita a través de Netlify.
